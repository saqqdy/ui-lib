# 安装

## yarn 安装

推荐使用 yarn 的方式安装，它能获得更好的安装速度。

```shell
# 通过yarn安装
yarn add ui-lib
# 或者通过npm安装
npm install -S ui-lib
```

## CDN <Badge text="即将支持"/>

在 index.html 的 head 里面引入

```html
<!-- 引入组件库 -->
<script src="https://unpkg.com/ui-lib/lib/index.js"></script>
```

::: tip
建议使用 CDN 引入 ui-lib 的用户在链接地址上锁定版本，以免将来升级时受到非兼容性更新的影响。
:::
